# _Son of Cod Seafood_

#### _A web application for a fish company that can grow their mailing list and display their product and also display news 4/28/2017_

#### By _**Chad Durkin**_

## Description

_A website displaying a company's product, along with news and marketing info, also it allows admins to register an account and log in to be able to further add more content or look at people who are signing up for their mailing list._

## Setup/Installation Requirements

* _Clone this repository https://github.com/Rafeekey/sonofcod
* _Open up in Microsoft Visual Studio 15_
* _Run Applicatoin_
